# [壹代理](http://bitbucket.org/CrazyBoyFeng/agentx1) #

提供具有良好兼容性、稳定性、易用性、针对性的 802.1X 自由代理 Linux 核心解决方案。 

## 简介 ##

项目是常见的私有 802.1X 网络认证双向代理的开源实现，能以中间人的身份在网络中协调服务器与客户端的认证过程。
